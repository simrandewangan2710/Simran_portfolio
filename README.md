# Simran Dewangan - Analytics Portfolio

A professional, single-page web portfolio showcasing my experience, skills, and projects in Data Analytics and Computer Science. It highlights actionable data insights derived from SQL, Excel, and Python tools, wrapped in a sleek, dark-themed interface.

## 🚀 Features

- **Dark Theme Aesthetics:** A modern, deep dark layout designed to emphasize high-contrast viewing and crisp typography.
- **Project Demonstrations:** Visual cards displaying direct links to repositories (Vrinda Store Data, Flipkart Web Scraper, Zepto Grocery Data Analytics).
- **Responsive Navigation:** Fixed navigation bar with seamless, auto-scrolling sections spanning Experience, Education, Resume, and Skills.
- **Scroll Animations:** Intersection observers built into Vanilla JavaScript to smoothly reveal components upward as they enter the viewport.

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture.
- **CSS3:** Flexbox, Grid layouts, custom styling, and variable CSS tokens.
- **Vanilla JavaScript:** DOM manipulations and Intersection Observer logic natively.
